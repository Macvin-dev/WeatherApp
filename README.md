iPhoneのGPSデータに基づいて、または手動で都市を検索して、現在位置の天気を確認するアプリです。
UITextFieldを使用して、ユーザーの入力実装。
URLSessionを使用してネットワークに接続、HTTPリクエスト、Encodable及びDecodableプロトコルでJSONをエンコード,デコード。
Core Locationを使用して、携帯電話GPSで現在位置実装。
